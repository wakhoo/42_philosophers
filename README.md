# 42 philosophers
> Concurrent Programming Project
> - Implemented solution to the Dining Philosophers problem using threads and mutexes
> - Managed shared resources and prevented race conditions, deadlocks, and data races
> - Developed precise timing mechanisms for thread synchronization
> - Built monitoring system to detect philosopher deaths in real-time

On Linux :
- cd `philo`
- Do `make` and `./philo number_of_philosophers time_to_die time_to_eat time_to_sleep [number_of_times_each_philosopher_must_eat]`
